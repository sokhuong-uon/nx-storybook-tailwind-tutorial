## Start the app

To start the development server run

```
nx serve web
```

Open your browser and navigate to http://localhost:4200/. Happy coding!

## Serve storybook

```
nx storybook ui
```

## Explore the Project Graph

Run `nx graph` to show the graph of the workspace.
